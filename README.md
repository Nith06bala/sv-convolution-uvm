# sv-convolution-uvm
SystemVerilog implementation of a streaming convolution engine with SRAM-based data fetch and data reuse optimization. Developed a complete UVM testbench including driver, monitor, scoreboard, constrained-random sequences, and functional coverage to verify boundary and streaming operations.
## Features

- SystemVerilog RTL
- SRAM-based data fetch
- Data reuse optimization
- Sliding window computation
- Boundary handling without zero padding
- UVM verification
- Scoreboard
